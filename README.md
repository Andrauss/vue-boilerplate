# vue-boilerplate

> An opinionated Vue.js boilerplate

## Progress
- [X] Webpack 2
- [X] ESLint
- [X] Vue.js
- [X] Vuex
- [X] Vuex Router Sync
- [X] Vue Router
- [X] Vue i18n
- [X] Fetch API
- [ ] Flow Type
- [X] Hotreload
- [X] SASS with Autoprefixer
- [X] AVA Tests
- [X] Code Coverage
- [ ] Offline Support
- [ ] Server-Side Rendering

## Installation

``` bash
### Install dependencies
yarn

### Start dev server with hot reload
yarn run dev

### Build everything
yarn run build

### Lint files
yarn run lint

### Run the tests
yarn run test

### Check the coverage
yarn run coverage
```
